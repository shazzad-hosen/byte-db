# ByteDB 🚀

ByteDB is a minimal database engine built from scratch in C++.

## Goals
- Understand how databases like MySQL/PostgreSQL work internally
- Learn low-level system design
- Practice C++ (memory, data structures, file handling)

## Features (Planned)
- In-memory storage
- Table creation
- Insert & query
- File persistence
- Indexing

## Tech Stack
- C++
- STL
